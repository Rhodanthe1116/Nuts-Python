# ⚡ 練習：觀念題

## 若要在 Python 中判斷各種情況，可用下列哪種語法？

1. `if...else if...else`
2. `if...elif...else`
3. `if...else if...elif`

## 下列哪個程式碼是對的？

1. ```python
   x = 10
   if x = 10:
       print("x等於10")
   else:
       print("x不等於10")
   ```
2. ```python
   x = 10
   if x == 10:
       print("x等於10")
   else:
       print("x不等於10")
   ```

## if 裡面包含另一個 if 稱作什麼？

## 下列程式碼會輸出什麼？

```python

    print(1)
