## 下列程式碼會輸出什麼？

```python
for x in range(10):
    if x % 2 == 0:
        break
    print(x)
```

## 下列程式碼會輸出什麼？

```python
for x in range(10):
    if x % 2 == 0:
        continue
    print(x)
```

## 下列哪段程式碼是對的？

1. ```python
   for x in range(10)
       print(x)
   ```
2. ```python
   for i in range(10):
       print(i)
   ```

### 請問下列程式輸出為何？

```python
x = 0
for i in range(n):
    for j in range(i, n):
        k = 1
        while k <= n:
            x = x + 1
            k *= 2
print(x)
```

1. n\(n+1\)\*sqrt\(log2\(𝑛\)\)
2. n2\(n+1\)/2
3. **n\(n+1\)\[ \(log2\(n\)\) + 1 \] / 2**
4. n\(n+1\)/2

### 請問執行下列程式的時候，哪一行可能永遠不會被執行到

### 若 n 為正整數，右側程式三個迴圈執行 完畢後 a 值將為何？

```python

for i in range(1, n+1):
    for j in range(i, n+1):
        for k in range(1, n+1):
            a = a + 1

```
